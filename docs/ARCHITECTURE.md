# Архитектура проекта Eurodib

## Быстрая справка

### Структура данных
- **SAP** = транзакции, цены, остатки, заказы
- **Strapi** = контент, медиа, SEO, маркетинг
- **Frontend** = получает данные из обоих источников

### Репозитории
1. `eurodib-frontend` (текущий) — Next.js приложение
2. `eurodib-strapi` (создать) — Strapi CMS
3. `sap-sync` (опционально) — синхронизация SAP → Strapi

### База данных
- **Только PostgreSQL** (без SQLite)

### Мультиязычность
- **EN + FR** с первого дня
- Использовать Strapi i18n plugin
